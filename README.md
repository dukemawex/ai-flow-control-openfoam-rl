# AI Flow Control (OpenFOAM + RL)

Reinforcement-learning-based active flow control using OpenFOAM simulations.

## Getting started

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m ai_flow_control_openfoam_rl
```

## Project structure

```
ai_flow_control_openfoam_rl/        # source package
tests/      # tests
requirements.txt
```
